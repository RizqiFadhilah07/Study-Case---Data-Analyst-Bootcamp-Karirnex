# 📊 Sales Dashboard – FMCG Data Analysis (Power BI)

## 📌 Overview

This project showcases an interactive **Sales Dashboard** built using **Power BI**, based on a dummy FMCG dataset provided by KarirNex as part of a **Data Analyst Bootcamp assignment**.

The project demonstrates an **end-to-end data analysis workflow**, starting from raw data cleaning using Python to building an insightful business dashboard.

---

## 🎯 Objectives

* Analyze sales performance across time, category, and channel
* Track key business metrics such as **Total Revenue, Total Orders, and Average Order Value (AOV)**
* Identify top-performing products and categories
* Monitor **Month-over-Month (MoM) growth**

---

## 🧹 Data Preparation (Python – Jupyter Notebook)

Data cleaning and preprocessing were performed using Python in a Jupyter Notebook:

* Handling **missing values**
* Removing **duplicate records**
* Standardizing **date formats**
* Cleaning inconsistent text (e.g., product names, categories)
* Ensuring correct **data types**
* Creating new features such as:

  * `Total Sales = Price × Quantity`

---

## 📊 Dashboard Features (Power BI)

* 📅 Time-based analysis using Month slicer
* 📈 Monthly revenue trend visualization
* 🛒 Sales breakdown by **Category & Channel**
* 🏆 Top-performing products
* 🔄 Interactive filters (Month, Channel, Category, Status)
* 📉 KPI indicators with **MoM Growth (%)**

---

## 📈 Key Metrics

* **Total Revenue**
* **Total Orders**
* **Average Order Value (AOV)**
* **Revenue Growth (MoM %)**

---

## 🛠️ Tools & Technologies

* **Python (Pandas, NumPy)** – Data cleaning & preprocessing
* **Jupyter Notebook (.ipynb)** – Data exploration & transformation
* **Power BI** – Data visualization & dashboarding
* **DAX (Data Analysis Expressions)** – Measures & calculations
* **Power Query** – Data transformation in Power BI

---

## 🚀 How to Use

1. Open the **Jupyter Notebook (`.ipynb`)** to see the data cleaning process
2. Open the `.pbix` file using Power BI Desktop
3. Explore the dashboard using slicers and filters

---

## 🧠 Key Insights

* Certain product categories consistently generate the highest revenue
* Sales performance differs across channels, indicating channel effectiveness
* Monthly trends highlight fluctuations and potential seasonality
* Growth indicators help quickly identify performance improvements or declines
